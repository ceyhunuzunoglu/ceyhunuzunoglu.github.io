    PTRLibraryHelper¶

      Inherits from NSObject
      Declared in PTRLibraryHelper.h

    Overview¶

    Static class including helper methods for Pointr Engine

    Class Methods¶

    alphabetOnlyLowercaseStringFromString:¶

    Strips string of any character other than alphabetic and returns(eg. numbers, spaces, symbols) Converts accents also (eg. é -> e) Used to match string from Pointr Cloud mainly

    + (NSString *)alphabetOnlyLowercaseStringFromString:(NSString *)*string*

    Parameters:

    string
    string to be stripped

    Returns:

    • NSString alphabetic representation of given string

    findAngleBetweenTwoPointsWithX1:y1:x2:y2:¶

    Finds angle between two points

    + (float)findAngleBetweenTwoPointsWithX1:(float)*x1* y1:(float)*y1* x2:(float)*x2* y2:(float)*y2*

    Parameters:

    x1
    x for 1st point

    y1
    y for 1st point

    x2
    x for 2nd point

    y2
    y for 2nd point

    Returns:

    • the angle of vector (x2,y2) -> (x1,y1) in radians, in coordinate system: (0,0) top-left, (1,1) bottom-right; angle 0 north, increasing clockwise

    isOldDevice¶

    Returns YES if device has poor CPU/memory. NO otherwise. This flag is used to determine whether low memory mode should be activated.

    + (BOOL)isOldDevice

    searchFriendlyString:¶

    Return search-friendly version of given string, lowercase and without accents (eg. è -> e) Used to create search tokens mainly

    + (NSString *_Nullable)searchFriendlyString:(NSString *)*str*

    Parameters:

    str
    string to make friendly

    Returns:

    • NSString search friendly string

    stringForLevel:¶

    Return string representation of level

    + (NSString *)stringForLevel:(int)*level*

    Parameters:

    level
    level

    Returns:

    • NSString string representation of level
